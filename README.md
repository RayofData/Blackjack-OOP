# Blackjack OOP Practice

A simple Blackjack project created to practice object-oriented programming in Python.

## Concepts Practiced

* Classes and objects
* Inheritance
* Methods
* Object composition
* Lists of objects
* Importing classes from separate files

## Classes

* `Card`: Represents a playing card.
* `Table`: Handles all Players
* `Player`: Base class for all players.
* `BettingPlayer`: Base class for Human and NPC players
* `HumanPlayer`: Handles human betting and actions.
* `NPCPlayer`: Handles computer-controlled actions.
* `Dealer`: Represents the dealer.

## Run the Project

```bash
python main.py
```

This project is currently a work in progress and is being used for OOP practice.
